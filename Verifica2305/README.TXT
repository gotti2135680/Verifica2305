Consegna esercizio
si vuole creare un progetto java dove dei container sono caratterizzati da un codice una stazza un
carico e una temperature. 
si vuole creare i metodi per l'aggiunta di un nuovo container, la ricerca di un container e la
rimozione di un container consegna diversa da quella inserita nel README.md perchè era sbagliata 
quella scritta qui è quella giusta.

Classe Container 
-dichiarazione delle variabili codice stazza carico e temperatura
-successivamente creazione del costruttore delle variabili 
- successivamente creazione dei get e set per tutte le variabili 
-creazione del tostring 
- creazione di un main per testare la classe 

Classe Deposito
-dichiarazione di una hashmap 
-creazione del costruttore per la hashmap
-metodo per aggiunta del container mediante il comando .put
-metodo per ricerca del container mediante il comando .get
-metodo per rimuovere il container mediante il comando .remove
